# Análise de Séries Temporais
Este projeto tem como finalidade realizar uma série de análises em conjuntos de dados e aplicar conceitos e ferramentas da ciência de dados.

Fonte: dados elaborados pela Alura.

Dentre os estudos realizados destaca-se:
- Tratamento e exploração dos dados
- Visualização dos dados
- Análise do aumento, aceleração e autocorrelação de vendas
- Decomposição dos componentes da série temporal aplicada as vendas de um produto
- Relação entre tabelas distintas e obtenção de insights
- Avaliação da taxa de churn
- Aplicação de coorte sobre clientes de uma campanha
